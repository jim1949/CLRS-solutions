# CLRS-solutions

Solutions to "Introduction to Algorithms, 3rd edition"

#How to compile it?

$ git clone git@github.com:jim1949/CLRS-solutions.git

$ cd CLRS-Solutions

$ pdflatex clrs.tex

And then you could find clrs.pdf in your directory.

#Find typos, errors or have other solutions?
